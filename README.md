# Placard — Poster Maker

Six centuries of art history. One poster for your shop.

An installable web app (PWA): 19 art movements, ~620 famous works as poster
canvases, and a photo-restyle engine that redesigns your uploaded photo in the
language of whichever movement and masterwork you pick.

**Use it / install it:** https://susiesu113-eng.github.io/placard/
On Android: Chrome will offer "Install app". On iPhone: Share → Add to Home Screen.

Works offline after the first visit.
